# Paradise Nursery

Welcome to Paradise Nursery, an online plant shop application built with React and Redux.

## Project Details
- **Project Name**: Paradise Nursery Shopping Application
- **Description**: A dynamic web application that allows users to browse a variety of houseplants, view details such as images, names, and prices, and manage a shopping cart. The application emphasizes interactive features such as a navigation bar, dynamically updating cart quantities, and calculating total costs as users manage their cart items.
- **Technologies Used**: React, Redux Toolkit, React Router DOM, Vite, CSS.

## Features
- Landing page with a visually appealing background and a "Get Started" button.
- Product listing page categorizing plants into three distinct categories, with Add to Cart functionality.
- dynamic Shopping Cart displaying total quantities, item costs, and options to increment, decrement, or remove items.
